## Как запустить проект

```
# Для того, чтобы собрать контейнеры при запуске, если раньше не были собраны
$ docker-compose up -d --build

# Для того, чтобы собрать контейнеры при запуске, если раньше не были собраны
$ docker-compose up -d
```
